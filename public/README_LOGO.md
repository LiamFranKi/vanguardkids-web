# 📸 Logo y Favicon

## 📁 Coloca tus archivos aquí:

### Logo
- **Archivo:** `logo.png` o `logo.svg`
- **Ubicación:** Esta misma carpeta (`public/`)
- **Tamaño recomendado:** 200x200px o más grande
- **Formato:** PNG con fondo transparente (recomendado) o SVG

### Favicon
- **Archivo:** `favicon.ico` o `favicon.png`
- **Ubicación:** Esta misma carpeta (`public/`)
- **Tamaño recomendado:** 32x32px o 64x64px

## ✅ Una vez que coloques los archivos:

1. El favicon se cargará automáticamente
2. Para el logo, necesitarás actualizar `components/Header.tsx` (ver INSTRUCCIONES_LOGO_FAVICON.md)
